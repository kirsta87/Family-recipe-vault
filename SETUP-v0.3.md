# Version 0.3 setup

1. In Apps Script, replace the current code with `APPS-SCRIPT-v0.3.js`.
2. Replace `CHANGE-THIS-TO-A-PRIVATE-FAMILY-KEY` with a private household phrase.
3. Save.
4. Deploy → Manage deployments → pencil icon → New version → Deploy.
5. Replace the website files in your cloned GitHub folder with this package.
6. In GitHub Desktop: commit, then Push origin.
7. On the live site, open Manage and paste:
   - published Google Sheet CSV URL
   - Apps Script web-app URL
   - the exact same family key
8. Save and test a rating or note.

The family key is stored only in each browser. It is not placed in the public GitHub files.
